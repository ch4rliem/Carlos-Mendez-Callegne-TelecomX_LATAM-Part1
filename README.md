
# 📡 Challenge Data Science: Análisis de Churn - Telecom X

Este proyecto forma parte del **Challenge de Alura Latam**, donde asumí el rol de Business Analyst para identificar los factores que impulsan la fuga de clientes (Churn) en una empresa de telecomunicaciones.

## 🚀 Resumen del Proyecto
El objetivo principal fue realizar un proceso **ETL** (Extracción, Transformación y Carga) sobre un dataset de clientes en formato JSON, procesar la información anidada y generar un análisis de impacto financiero.

## 📊 Principales Hallazgos
- **Impacto Económico**: Se identificó una pérdida acumulada de **$2.86M USD**.
- **Factor Tecnológico**: Los usuarios de **Fibra Óptica** tienen una tasa de evasión del **41.8%**, significativamente superior al DSL.
- **Factor Contractual**: El contrato "Mes a Mes" es el mayor disparador de fugas (41.3% de abandono).
- **Perfil de Riesgo**: Los clientes que abandonan el servicio pagan, en promedio, **$13 USD más** al mes que los clientes leales.
